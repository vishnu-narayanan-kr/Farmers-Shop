How to use this Application

The Home page shows the user a general idea about QuÃ©bec Farmer's Shop. The User can choose a list of Products from that's availabe to their cart. Once finalized, they could go to the purchase section to confirm their purchase. The cart data would be preserved even if the user leaves or closes the Application in between. For the purchase, the user has to login, and present their credentials. If the user is not already registered, they could do the registering, and confirm their purchase. When the user goes back, the updated inventory will be reflected in Products page. The following section helps you reset the Application data, if needed.

Note: This Application is only a demo, and not all functionalities are throughly implemented, and could also be buggy

Resetting the app would delete all data stored, which includes

  1. Username and Passwords
  2. Cookies
  3. Saved Cart Items
  4. Resets Inventory Items
